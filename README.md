### 📘 Conocimientos técnicos destacados
- **Arquitectura de Microservicios:** Service Discovery (Netflix Eureka), centralización de configuración (Spring Cloud Config) y comunicación síncrona declarativa (OpenFeign).
- **Infraestructura y Despliegue:** Contenedorización de entornos complejos mediante **Docker** y **Docker Compose** (gestión de redes internas y volúmenes persistentes).
- **Seguridad:** Autenticación y autorización robusta mediante **JWT** (JSON Web Tokens) y gestión de roles.
- **Diseño de APIs REST:** Uso del patrón DTO (con **MapStruct**), validaciones, y documentación interactiva de endpoints con **Swagger/OpenAPI**.
- **Manejo de Errores:** Implementación de `@RestControllerAdvice` y decodificadores personalizados para respuestas globales estandarizadas.
- **Persistencia de Datos:** Modelado de bases de datos relacionales y mapeo objeto-relacional (JPA/Hibernate).
